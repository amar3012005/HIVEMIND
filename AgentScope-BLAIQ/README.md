# AgentScope-BLAIQ

Fresh API-first multi-agent runtime for BLAIQ with:

- strategist, research, vangogh, and governance agents
- sequential, parallel, and hybrid workflows
- SSE streaming
- Postgres + Redis persistence
- Coolify/Hetzner Docker Compose deployment

This project intentionally excludes the graph knowledge-base agent from v1 orchestration while preserving extension points for it.
