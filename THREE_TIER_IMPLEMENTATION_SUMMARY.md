## API Endpoints

### POST /api/search/quick
Fast semantic search for immediate results.
- **Limit:** 10 results (configurable)
- **Excludes:** Expired content by default
- **Depth:** Shallow
- **Target Latency:** <100ms

### POST /api/search/panorama
Comprehensive search including historical/expired content.
- **Limit:** 50 results (configurable)
- **Includes:** Expired and historical content
- **Features:** Temporal categorization, timeline view
- **Target Latency:** <500ms

### POST /api/search/insight
Deep multi-dimensional analysis with LLM-powered insights.
- **Sub-queries:** 3-7 generated by LLM
- **Features:** Entity extraction, relationship chains, semantic facts
- **Requirements:** Groq API key
- **Target Latency:** <3s

### POST /api/search/compare
Compare results across all three tiers.
- **Returns:** Performance metrics for each tier
- **Use Case:** A/B testing, tier selection

## Dependencies

### Required
- Existing vector store (Qdrant)
- Existing graph store (Prisma/Apache AGE)
- Node.js built-in modules (http, crypto, etc.)

### Optional (for InsightForge)
- Groq API key for LLM-powered analysis
- Model: `llama-3.3-70b-versatile`

## Configuration

### Environment Variables
```bash
# Required for InsightForge
GROQ_API_KEY=your-groq-api-key
GROQ_INFERENCE_MODEL=llama-3.3-70b-versatile

# Optional overrides
HIVEMIND_SEARCH_DEFAULT_LIMIT=20
HIVEMIND_SEARCH_MAX_LIMIT=100
```

### Default Configuration
```javascript
CONFIG = {
  limits: {
    quickSearch: 10,
    panoramaSearch: 50,
    insightForge: 30
  },
  thresholds: {
    quickSearch: 0.3,
    panoramaSearch: 0.2,
    insightForge: 0.25
  },
  weights: {
    quickSearch: { vector: 0.7, keyword: 0.2, graph: 0.1 },
    panoramaSearch: { vector: 0.5, keyword: 0.3, graph: 0.2 },
    insightForge: { vector: 0.4, keyword: 0.2, graph: 0.3, insight: 0.1 }
  }
}
```

## Usage Examples

### QuickSearch
```bash
curl -X POST http://localhost:3000/api/search/quick \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"query": "machine learning", "limit": 5}'
```

### PanoramaSearch
```bash
curl -X POST http://localhost:3000/api/search/panorama \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "query": "project history",
    "include_expired": true,
    "include_historical": true
```
