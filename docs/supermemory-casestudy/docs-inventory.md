## Why These Pages Matter

The broader docs surface matters because the concepts pages alone do not fully explain:

- how ingestion is exposed in product/API form
- how progress and status are tracked
- how connectors feed the same system
- how user profiles are operationalized
- how the Memory Router changes the integration model

That broader surface is what turns the conceptual model into a usable architecture.
