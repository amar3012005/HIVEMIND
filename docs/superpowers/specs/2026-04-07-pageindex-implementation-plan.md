## Overview

### Purpose

Add PageIndex as an **OPTIONAL optimization layer** (never a dependency) that enables:
1. Hierarchical memory organization (topics/projects tree)
2. Sub-10ms topic-scoped navigation
3. Auto-classification of new memories during ingestion
4. Hybrid search fusion (PageIndex + VectorDB)
5. Graceful fallback to PostgreSQL + VectorDB if PageIndex fails

### Key Design Decisions (from Brainstorming)

| Decision | Choice | Rationale |
|----------|--------|-----------|
| **Storage** | PostgreSQL table | Leverages existing DB, no new infrastructure |
| **Content** | Memory IDs only | Full memories stay in PostgreSQL, no duplication |
| **Depth** | Max 4 levels | Prevents over-nesting, keeps traversal fast |
| **Classification** | LLM async queue | Doesn't block ingestion, retries on failure |
| **Cross-ref** | Multiple parent nodes | Single memory can live under multiple topics |
| **Fallback** | Silent degradation | System works without PageIndex enabled |

### Fallback Guarantees

```
┌─────────────────────────────────────────────────────────────┐
│  PageIndex Available → Use for navigation + hybrid search   │
│                          ↓                                  │
│  PageIndex Fails     → Log warning, continue with VectorDB  │
│                          ↓                                  │
│  VectorDB Fails      → Fall back to PostgreSQL lexical      │
│                          ↓                                  │
│  All Optional Layers → System still ingests + retrieves     │
└─────────────────────────────────────────────────────────────┘
```

**Critical Constraint**: PageIndex must NEVER be a blocking dependency. All operations must:
- Have timeout limits (100ms for classification, 50ms for search lookup)
- Catch and log errors without throwing
- Return gracefully to fallback paths

### Task 1.3: Backfill Script (Async, Idempotent)

**File**: `core/scripts/backfill-pageindex.js` (NEW)

**Purpose**: Populate PageIndex with existing memories from database

**Algorithm**:
```javascript
async function backfillPageIndex() {
  // 1. Create root node per user
  const users = await prisma.user.findMany({ select: ['id'] });

  for (const user of users) {
    const rootNode = await pageIndexService.ensureRootNode(user.id);

    // 2. Extract topics from existing memories
    const memories = await prisma.memory.findMany({
      where: { userId: user.id, deletedAt: null },
      select: ['id', 'project', 'tags', 'content', 'memoryType']
    });

    // 3. Group by project/tags → create nodes
    const nodeMap = new Map(); // topic → nodeId

    for (const memory of memories) {
      const topic = memory.project || extractTopicFromTags(memory.tags);
      if (!topic) continue;

      // Create node if not exists
      let nodeId = nodeMap.get(topic);
      if (!nodeId) {
        const node = await pageIndexService.createNode({
          userId: user.id,
          parentId: rootNode.id,
          label: topic,
          nodeType: 'category'
        });
        nodeId = node.id;
        nodeMap.set(topic, nodeId);
      }

      // Assign memory to node
      await pageIndexService.assignMemoryToNode(nodeId, memory.id);
    }
  }
}
```

**Idempotency Guarantees**:
- Check if memory already assigned → skip if exists
- Use `upsert` for node creation (find by path, create if not exists)
- Safe to re-run multiple times

**Retry Safety**:
- Batch processing (100 memories per batch)
- Checkpoint progress in `page_index_backfill_state` table
- Resume from last checkpoint on failure

```javascript
class PageIndexClassifier {
  constructor(options = {}) {
    this.groqApiKey = options.groqApiKey || process.env.GROQ_API_KEY;
    this.timeout = options.timeout || 100; // ms
    this.maxDepth = options.maxDepth || 4;
  }

  async classify(memoryContent, hierarchyTree) {
    // Timeout protection: return null if >100ms
    const controller = new AbortController();
    const timeoutId = setTimeout(() => controller.abort(), this.timeout);

    try {
      const prompt = buildClassificationPrompt(memoryContent, hierarchyTree);

      const response = await fetch('https://api.groq.com/openai/v1/chat/completions', {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${this.groqApiKey}`,
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({
          model: 'llama-3.1-8b-instant', // Fast model for classification
          messages: [
            {
              role: 'system',
              content: `You classify memories into hierarchical topic nodes.
Output JSON: { nodeId: string, confidence: number, path: string }
If no suitable node exists, return { nodeId: null, confidence: 0, shouldCreateNode: false }`
            },
            { role: 'user', content: prompt }
          ],
          temperature: 0,
          max_tokens: 200
        }),
        signal: controller.signal
      });

      const data = await response.json();
      return parseClassificationOutput(data);
    } catch (err) {
      if (err.name === 'AbortError') {
        console.warn('[PageIndexClassifier] Timeout exceeded, skipping classification');
        return null;
      }
      console.warn('[PageIndexClassifier] Classification failed:', err.message);
      return null; // Fallback: skip classification, retry async
    } finally {
      clearTimeout(timeoutId);
    }
  }
}
```

### Task 2.2: Async Classification Queue

**File**: `core/src/services/pageindex-queue.js` (NEW)

**Purpose**: Non-blocking classification that doesn't delay ingestion

**Architecture**:
```
Ingestion Pipeline → Queue classification job → Return immediately
                         ↓
              Background worker processes
                         ↓
              Assign memory to node (async)
```

**Implementation**:
```javascript
class PageIndexClassificationQueue {
  constructor(options = {}) {
    this.queue = new ArrayDeque();
    this.processing = false;
    this.retryAttempts = new Map(); // jobId → attempts
    this.maxRetries = options.maxRetries || 3;
  }

  async enqueue(memoryId, userId, memoryContent) {
    const job = {
      id: crypto.randomUUID(),
      memoryId,
      userId,
      memoryContent,
      createdAt: Date.now(),
      status: 'pending'
    };

    this.queue.push(job);
    this.processQueue(); // Fire and forget
    return job.id;
  }

  async processQueue() {
    if (this.processing) return;
    this.processing = true;

    while (this.queue.length > 0) {
      const job = this.queue.shift();
      try {
        await this.processJob(job);
      } catch (err) {
        await this.handleJobFailure(job, err);
      }
    }

    this.processing = false;
  }

  async processJob(job) {
    // Get current hierarchy tree
    const tree = await pageIndexService.getTree(job.userId);

    // Classify
    const classification = await classifier.classify(job.memoryContent, tree);

    if (classification?.nodeId) {
      await pageIndexService.assignMemoryToNode(classification.nodeId, job.memoryId);
    } else if (classification?.shouldCreateNode) {
      const newNode = await pageIndexService.createNode({
        userId: job.userId,
        parentId: findBestParentNode(tree, classification.suggestedPath),
        label: classification.suggestedLabel,
        nodeType: 'topic'
      });
      await pageIndexService.assignMemoryToNode(newNode.id, job.memoryId);
    }
  }

  async handleJobFailure(job, err) {
    const attempts = this.retryAttempts.get(job.id) || 0;
    if (attempts < this.maxRetries) {
      this.retryAttempts.set(job.id, attempts + 1);
      this.queue.push(job); // Re-queue for retry
    } else {
      console.error('[PageIndexQueue] Job failed after max retries:', job.id);
    }
  }
}
```

```javascript
async function createNodeWithDepthLimit(userId, parentId, label, nodeType) {
  const parent = await pageIndexService.findNodeById(parentId);

  if (!parent) {
    throw new Error('Parent node not found');
  }

  if (parent.depth >= 4) {
    // Can't go deeper — assign to parent instead
    console.warn('[PageIndex] Max depth reached, assigning to parent:', parent.path);
    return parent;
  }

  const newPath = `${parent.path}/${label.toLowerCase().replace(/\s+/g, '-')}`;

  // Check if path already exists
  const existing = await pageIndexService.findNodeByPath(userId, newPath);
  if (existing) {
    return existing;
  }

  // Create new node
  return await pageIndexService.createNode({
    userId,
    parentId,
    label,
    nodeType,
    depth: parent.depth + 1,
    path: newPath
  });
}
```

### Task 3.1: PageIndexSearcher with Timeout + Fallback

**File**: `core/src/search/pageindex-searcher.js` (NEW)

**Purpose**: Fast hierarchical lookup (<50ms target)

**Implementation**:
```javascript
class PageIndexSearcher {
  constructor(options = {}) {
    this.timeout = options.timeout || 50; // ms
    this.pageIndexService = options.pageIndexService;
  }

  async search(query, options = {}) {
    const { userId, orgId, topicPath, limit = 20 } = options;

    // Timeout protection
    const controller = new AbortController();
    const timeoutId = setTimeout(() => controller.abort(), this.timeout);

    try {
      // If topic path specified → direct tree traversal
      if (topicPath) {
        const node = await this.pageIndexService.findNodeByPath(userId, topicPath);
        if (node) {
          return {
            source: 'pageindex',
            memoryIds: node.memoryIds.slice(0, limit),
            nodePath: node.path,
            durationMs: Date.now() - startTime
          };
        }
      }

      // Otherwise: keyword match on node labels
      const tree = await this.pageIndexService.getTree(userId);
      const matchingNodes = findNodesByKeyword(tree, query);

      const allMemoryIds = [];
      for (const node of matchingNodes.slice(0, 5)) {
        allMemoryIds.push(...node.memoryIds);
      }

      return {
        source: 'pageindex',
        memoryIds: [...new Set(allMemoryIds)].slice(0, limit),
        durationMs: Date.now() - startTime
      };
    } catch (err) {
      console.warn('[PageIndexSearcher] Search failed, falling back:', err.message);
      return {
        source: 'fallback',
        memoryIds: [],
        error: err.message
      };
    } finally {
      clearTimeout(timeoutId);
    }
  }
}

function findNodesByKeyword(tree, query) {
  const results = [];
  const queryLower = query.toLowerCase();

  function traverse(node) {
    if (node.label.toLowerCase().includes(queryLower)) {
      results.push(node);
    }
    if (node.children) {
      node.children.forEach(traverse);
    }
  }

  tree.forEach(traverse);
  return results.sort((a, b) => b.memoryCount - a.memoryCount);
}
```

```javascript
async function hybridSearchWithPageIndex(query, options) {
  const { userId, orgId, usePageIndex = true } = options;

  const searchPromises = [];

  // PageIndex (optional, fast)
  if (usePageIndex) {
    searchPromises.push(
      pageIndexSearcher.search(query, { userId, orgId })
        .then(r => r.memoryIds.map(id => ({ id, source: 'pageindex' })))
        .catch(() => []) // Silent fallback
    );
  }

  // VectorDB (semantic)
  searchPromises.push(
    qdrantClient.searchMemories({ query, userId, orgId, limit: 20 })
      .then(r => r.map(p => ({ id: p.id, score: p.score, source: 'vector' })))
      .catch(() => [])
  );

  // PostgreSQL (lexical fallback)
  searchPromises.push(
    prisma.memory.findMany({ /* keyword search */ })
      .then(r => r.map(m => ({ id: m.id, source: 'lexical' })))
      .catch(() => [])
  );

  const resultSets = await Promise.all(searchPromises);
  return fuseResults(resultSets, { strategy: 'rrf', k: 60 });
}
```

## Phase 4: Auto-Evolution

### Task 4.1: CSI Agent for Hierarchy Restructuring

**File**: `core/src/agents/pageindex-evolver.js` (NEW)

**Purpose**: Periodic analysis and optimization of hierarchy structure

**Agent Type**: CSI (Faraday) — runs daily/weekly

**Responsibilities**:
1. Analyze node sizes (detect imbalances)
2. Identify sparse nodes (<5 memories)
3. Identify overloaded nodes (>100 memories)
4. Suggest restructuring (merge/split)

**Implementation**:
```javascript
class PageIndexEvolver {
  async analyzeAndRestructure(userId) {
    const tree = await pageIndexService.getTree(userId);
    const recommendations = [];

    // Find sparse nodes
    const sparseNodes = findNodesWithThreshold(tree, (n) => n.memoryCount < 5);
    for (const node of sparseNodes) {
      if (!node.customBranch && Date.now() - node.updatedAt > 30 * 24 * 60 * 60 * 1000) {
        recommendations.push({
          action: 'merge',
          nodeId: node.id,
          suggestion: `Merge "${node.label}" into parent "${node.parent?.label}"`
        });
      }
    }

    // Find overloaded nodes
    const overloadedNodes = findNodesWithThreshold(tree, (n) => n.memoryCount > 100);
    for (const node of overloadedNodes) {
      const suggestedSplits = analyzeMemoryClustering(node.memoryIds);
      if (suggestedSplits.length > 1) {
        recommendations.push({
          action: 'split',
          nodeId: node.id,
          suggestion: `Split "${node.label}" into: ${suggestedSplits.map(s => s.label).join(', ')}`
        });
      }
    }

    // Apply high-confidence recommendations automatically
    for (const rec of recommendations.filter(r => r.confidence > 0.9)) {
      if (rec.action === 'merge') {
        await mergeNodeIntoParent(rec.nodeId);
      } else if (rec.action === 'split') {
        await splitNode(rec.nodeId, rec.suggestedSplits);
      }
    }

    // Present low-confidence recommendations for manual review
    const pendingRecommendations = recommendations.filter(r => r.confidence <= 0.9);
    if (pendingRecommendations.length > 0) {
      await notifyUser(userId, {
        type: 'pageindex-restructure',
        recommendations: pendingRecommendations
      });
    }

    return { analyzedAt: new Date(), recommendations };
  }
}
```

```javascript
async function pruneSparseNodes(userId) {
  const thirtyDaysAgo = new Date(Date.now() - 30 * 24 * 60 * 60 * 1000);

  const sparseNodes = await prisma.pageIndexNode.findMany({
    where: {
      userId,
      memoryCount: { lt: 5 },
      customBranch: false, // Don't prune user-created branches
      updatedAt: { lt: thirtyDaysAgo },
      deletedAt: null
    }
  });

  for (const node of sparseNodes) {
    // Move memories to parent
    if (node.parentId) {
      const parent = await prisma.pageIndexNode.findUnique({
        where: { id: node.parentId }
      });

      // Merge memory IDs
      const mergedMemoryIds = [...new Set([...parent.memoryIds, ...node.memoryIds])];

      await prisma.pageIndexNode.update({
        where: { id: node.parentId },
        data: {
          memoryIds: mergedMemoryIds,
          memoryCount: mergedMemoryIds.length
        }
      });
    }

    // Soft delete the sparse node
    await prisma.pageIndexNode.update({
      where: { id: node.id },
      data: {
        deletedAt: new Date(),
        lastPrunedAt: new Date()
      }
    });
  }
}
```

```javascript
async function splitOverloadedNode(nodeId) {
  const node = await prisma.pageIndexNode.findUnique({
    where: { id: nodeId },
    include: { user: true }
  });

  if (node.memoryCount <= 100) {
    return; // No split needed
  }

  // Fetch memories for clustering analysis
  const memories = await prisma.memory.findMany({
    where: {
      id: { in: node.memoryIds },
      userId: node.userId
    },
    select: ['id', 'content', 'tags', 'project']
  });

  // Cluster by tags/embeddings
  const clusters = clusterMemoriesBySimilarity(memories);

  if (clusters.length < 2) {
    return; // Can't split meaningfully
  }

  // Create child nodes for each cluster
  for (const cluster of clusters) {
    const childNode = await prisma.pageIndexNode.create({
      data: {
        userId: node.userId,
        parentId: node.id,
        label: cluster.suggestedLabel,
        nodeType: 'subtopic',
        depth: node.depth + 1,
        path: `${node.path}/${cluster.suggestedLabel.toLowerCase()}`,
        memoryIds: cluster.memoryIds,
        memoryCount: cluster.memoryIds.length,
        autoGenerated: true
      }
    });

    console.log('[PageIndex] Created split node:', childNode.path);
  }

  // Clear parent memory IDs (now in children)
  await prisma.pageIndexNode.update({
    where: { id: nodeId },
    data: {
      memoryIds: [],
      memoryCount: 0
    }
  });
}

function clusterMemoriesBySimilarity(memories) {
  // Simple approach: cluster by most common tag
  const tagClusters = new Map();

  for (const memory of memories) {
    const primaryTag = memory.tags[0] || 'uncategorized';
    if (!tagClusters.has(primaryTag)) {
      tagClusters.set(primaryTag, []);
    }
    tagClusters.get(primaryTag).push(memory.id);
  }

  return Array.from(tagClusters.entries()).map(([tag, memoryIds]) => ({
    suggestedLabel: tag,
    memoryIds,
    confidence: memoryIds.length / memories.length
  }));
}
```

## Risk Mitigation

| Risk | Mitigation |
|------|------------|
| **PageIndex becomes a dependency** | All methods have `Safe` variants with silent fallback |
| **Classification blocks ingestion** | Async queue with fire-and-forget pattern |
| **Tree becomes too deep** | Hard limit at depth=4, enforced at creation |
| **Data duplication** | Store only memory IDs, never content |
| **Backfill corrupts data** | Idempotent operations, checkpoint-based resume |
| **Search performance degrades** | 50ms timeout, fallback to VectorDB |
| **Auto-evolution makes bad decisions** | High-confidence threshold (0.9) for auto-changes |
